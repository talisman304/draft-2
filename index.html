Is this for you?

This is positioned as advanced, intensive training. If you are very new, you may prefer a fundamentals course first.

Best fit if you:

Want to refine an existing skill set into a more predictable, aesthetic-driven workflow.

  • Have placed composite restorations and want stronger smile-design translation.
  • Want repeatable finishing, texture and gloss control.
  • Prefer hands-on, clinically relevant drills over theory-heavy teaching.

Consider another route if you:

Are looking for a first introduction to composite veneers.

  • Have never layered composites or built anterior anatomy.
  • Need fundamentals of isolation, bonding and incremental build-up.
  • Prefer a slower pace with more introductory demos.

If you are unsure, request guidance via the enquiry form—include your typical case mix and experience level.
